Huly MCP

MCP server for connecting AI agents to Huly workspaces through project, issue, document, contact, calendar, notification, search, and workspace tools.

Connect on Smithery GitHub npm

Hosted Use

The hosted MCP endpoint is managed through Smithery. Users provide Huly connection settings in Smithery, and Smithery forwards them to the server as request headers.

  • Huly URL: base URL for the Huly instance.
  • Workspace: Huly workspace identifier.
  • API token: Huly API token for that workspace.

Local Use

Local stdio usage is available from npm for clients that run MCP servers directly.

npx -y @firfi/huly-mcp@latest
