Huly MCP @firfi/huly-mcp

Give coding agents a real handle on Huly.

Huly MCP is a TypeScript Model Context Protocol server for reading and changing Huly workspace state from the agent tools you already use.

GitHub source, issues, releases npm npm install with npx Huly the workspace platform
npx -y @firfi/huly-mcp@latest

Manifesto

Work in the source of truth.

Agents should inspect and update the live Huly workspace, not copy tasks into a side system.

Expose the whole work graph.

Issues, projects, documents, contacts, cards, comments, files, calendars, activity, and tests belong in one tool surface.

Keep setup boring.

A standard MCP client should be able to run the server with npm, a Huly URL, workspace, and credentials.

Prefer readable operations.

Tools should accept human identifiers where possible and return enough context for the next agent step.

Treat credentials as volatile.

Runtime diagnostics are useful; leaking emails, tokens, passwords, URL secrets, or credential headers is not.

Let teams choose scope.

Tool categories can be filtered so an agent sees the part of Huly that matches the job at hand.
